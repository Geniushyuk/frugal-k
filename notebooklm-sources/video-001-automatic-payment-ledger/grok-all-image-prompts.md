# Grok 전달용 — 알뜰한 K씨 36컷 이미지 생성 명령서

## 목표
알뜰한 K씨 영상 001의 정지 이미지 36장을 만든다. 각 이미지는 이후 5초 영상 컷으로 움직일 키프레임이다.

## 반드시 유지할 캐릭터
기준 이미지가 있으면 먼저 업로드한다.

```text
drafts-private/assets/k-character-sheet-fixed.png
```

K씨 고정 요소:
- 둥근 얼굴
- 짧고 대충 빗은 검은 머리
- 검은 사각 안경
- 하얀 셔츠
- 초록/카키 가디건
- 검은 바지
- 거친 검은 펜선
- 색연필 질감
- 영수증, 계산기, 가계부, 절약 도장 소품

## 전체 스타일
한국 생활비 장부 느낌의 손그림 애니메이션 키프레임, 순백색 흰색 배경 고정, 배경색 변화 금지, 거친 검은 펜선, 색연필 질감, 하얀 셔츠, 초록/카키 가디건, 검은 바지, 검은 사각 안경. 실제 브랜드 로고, 카드사 로고, 앱 로고, 서비스 로고 금지. 긴 글자 금지. 한 이미지는 16:9 단일 컷. 한 컷 안에 여러 장면을 분할하지 말 것.

## 제작 규칙
- CUT 01부터 CUT 36까지 각각 별도 이미지로 생성한다.
- 파일명은 반드시 `frame-01.png`부터 `frame-36.png`까지 맞춘다.
- 각 컷은 5초 영상으로 움직일 예정이라, 프롬프트의 샷 사이즈와 다음 컷 연결을 반영한다.
- 브랜드명은 숫자 설명이 필요한 곳에만 텍스트로 쓰고, 실제 로고는 절대 넣지 않는다.
- 스마트폰/앱 화면은 추상 UI로 표현하고 개인정보는 넣지 않는다.
- 모든 컷의 배경은 같은 순백색 흰색으로 유지한다. 다른 배경색이나 종이색 톤으로 바꾸지 않는다.
- 구도는 컷의 감정과 핵심 오브젝트가 가장 잘 보이게 잡는다.
